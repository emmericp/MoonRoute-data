\subsection{Detailed Description}
R\+T\+E devargs\+: list of devices and their user arguments

This file stores a list of devices and their arguments given by the user when a D\+P\+D\+K application is started. These devices can be P\+C\+I devices or virtual devices. These devices are stored at startup in a list of \hyperlink{structrte__devargs}{rte\+\_\+devargs} structures.
