Function to set a callback to be triggered when us microseconds have expired. Accuracy of timing to the microsecond is not guaranteed. The alarm function will not be called {\itshape before} the requested time, but may be called a short period of time afterwards. The alarm handler will be called only once. There is no need to call \char`\"{}rte\+\_\+eal\+\_\+alarm\+\_\+cancel\char`\"{} from within the callback function.
