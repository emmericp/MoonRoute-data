\subsection{Detailed Description}
Structure that stores a device given by the user with its arguments

A user device is a physical or a virtual device given by the user to the D\+P\+D\+K application at startup through command line arguments.

The structure stores the configuration of the device, its P\+C\+I identifier if it\textquotesingle{}s a P\+C\+I device or the driver name if it\textquotesingle{}s a virtual device.
