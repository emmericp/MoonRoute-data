Initialize the Environment Abstraction Layer (E\+A\+L).

This function is to be executed on the M\+A\+S\+T\+E\+R lcore only, as soon as possible in the application\textquotesingle{}s main() function.

The function finishes the initialization process that was started during boot (in case of baremetal) or before main() is called (in case of linuxapp). It puts the S\+L\+A\+V\+E lcores in the W\+A\+I\+T state.

When the multi-\/partition feature is supported, depending on the configuration (if C\+O\+N\+F\+I\+G\+\_\+\+R\+T\+E\+\_\+\+E\+A\+L\+\_\+\+M\+A\+I\+N\+\_\+\+P\+A\+R\+T\+I\+T\+I\+O\+N is disabled), this function waits to ensure that the magic number is set before returning. See also the \hyperlink{rte__eal_8h_aab31145149c10993848d863b34d3d96f}{rte\+\_\+eal\+\_\+get\+\_\+configuration()} function. Note\+: This behavior may change in the future.
